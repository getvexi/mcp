# @getvexi/mcp

Vexi MCP Server — Structured business data for AI agents.

## Install

Add to your MCP client config:

```json
{
  "mcpServers": {
    "vexi": {
      "command": "npx",
      "args": ["-y", "@getvexi/mcp"],
      "env": {
        "VEXI_API_KEY": "vxi_live_..."
      }
    }
  }
}
```
